Updated [Structured data](/spec/seo/structured-data/) to reflect that Google retired the FAQ rich result in 2026. `FAQPage` is still valid schema.org vocabulary, but it no longer produces a search feature and no answer engine has confirmed it favours the markup over rendered HTML — so add it only for genuine, visible Q-and-A content, never for SERP or "GEO" gain.
